## 2. 教材制作の方針

### テキスト教材（ブラッシュアップ）

**原則：既存テキストの質と量を一切損なわず、学習体験を向上させる**

具体的な改善点：

1. **セクション対応の明確化**：試験の12セクションと教材の対応を明示し、学習の優先度を可視化
2. **弱点セクションの新規テキスト作成**：知識ガイド3の内容を、既存テキスト（第1〜3章）と同レベルの学習ガイドに再構成
3. **マイクロラーニング対応**：1トピック5〜10分で完結する「学習ユニット」に分割。短時間×複数回の学習スタイルに最適化
4. **記憶定着の仕掛け**：語呂合わせ、図解、対比表、ストーリー化を全セクションに追加
5. **実務との接続**：Ryuheiさんの調達実務経験と結びつけた説明を加え、理解と記憶を促進

### 演習問題

**原則：実際の試験形式を再現し、あらゆる角度から大量に出題**

1. **試験形式の忠実な再現**：選択式（穴埋め・複数選択）、90分90問のペースを意識
2. **セクション別問題集**：全12セクションをカバー、弱点セクションは特に厚く
3. **難易度3段階**：基礎確認→標準→応用（ひっかけ問題・複合問題）
4. **大量の問題数**：各セクション最低30問、弱点セクションは50問以上。合計500問以上を目標
5. **解説の充実**：正答だけでなく「なぜ他の選択肢が誤りか」まで解説

## 4. 学習設計（マイクロラーニング構成）

Ryuheiさんの「まとまった時間が取れない」という制約に最適化：

### 1日の学習サイクル（例）

| 時間帯 | 所要時間 | 内容 |
|---|---|---|
| 朝（通勤等） | 5〜10分 | フラッシュカードで暗記チェック |
| 昼休み | 10〜15分 | テキスト1ユニット読み込み |
| 午後の隙間 | 5分 | 演習問題5問（クイック演習） |
| 夜（帰宅後） | 15〜20分 | 演習問題15問 + 間違い復習 |

**1日合計：35〜50分 × 毎日継続**

### テキストの「学習ユニット」構成

各ユニットは以下の構造で統一：

```
【ユニットタイトル】（所要時間：8分）
├── 1. 導入（なぜこれが重要？）: 1分
├── 2. コア知識（図解・対比表付き）: 4分
├── 3. 暗記ポイント（語呂合わせ等）: 1分
├── 4. 確認クイズ（3問）: 2分
└── 次のユニットへの橋渡し
```
