# 調達マスターへの道

CPP b級試験対策アプリ - 実務シミュレーション型＋試験対策型の学習Webアプリ

## 🌐 アプリケーションURL

**https://brassinosteroidmr-lang.github.io/cpp-quiz-app/**

## ✨ 主な機能

### 🎯 実務シミュレーションモード
- **リアルな業務シナリオ**: 緊急調達、品質問題対応、コスト削減プロジェクトなど、実際の業務場面を再現
- **意思決定トレーニング**: 複数の選択肢から最適な判断を選び、その結果を即座にフィードバック
- **ストーリー進行**: 選択によって展開が変わる、没入感のある学習体験
- **3つのシナリオ**を収録（レベル別に解放）

### 📝 試験対策モード（新機能！）
- **145問の本格的な試験問題**を収録
- **全11章に対応**（第1章〜第11章）
- **四択形式**で実際の試験に準拠
- **各問題に詳しい解説付き**
- **わかりやすい正解・不正解表示**
- 章ごとの学習進捗管理
- 正答率の記録・分析

### 🎮 ゲーミフィケーション
- **レベルシステム**: 経験値を獲得してレベルアップ
- **昇進システム**: 新人 → 主任 → 課長 → 部長とキャリアアップ
- **実績バッジ**: 様々な達成条件をクリアしてバッジを獲得

### 📱 PWA対応
- **オフライン対応**: 一度読み込めば、インターネットなしでも学習可能
- **インストール可能**: スマホ・PCのホーム画面に追加して、アプリのように使用
- **レスポンシブデザイン**: スマホ・タブレット・PCあらゆるデバイスに対応

## 🚀 使い方

### Webで使う
1. https://brassinosteroidmr-lang.github.io/cpp-quiz-app/ にアクセス
2. 「新しく始める」をクリック
3. モードを選択：
   - 🎯 **実務シミュレーション**：ストーリーで学ぶ
   - 📝 **試験対策モード**：問題演習で学ぶ
4. 学習開始！

### ローカルで起動
```bash
# Pythonの場合
python -m http.server 8000

# Node.jsの場合
npx http-server
```

ブラウザで `http://localhost:8000` にアクセス

### スマホでアプリのように使う
ブラウザのメニューから「ホーム画面に追加」を選択

## 💡 特徴

### 詳しい解説
各問題に以下を含む解説を用意：
- ✅ 正解の理由
- 📚 関連知識の補足
- 💡 覚え方のコツ
- 🎯 試験のポイント

### わかりやすい判定表示
- **正解時**：大きな緑色の✓マークと「正解！」
- **不正解時**：大きな赤色の✗マークと「不正解」
- シンプルでわかりやすい演出

### 学習モチベーション維持
- 問題を解いて経験値を獲得
- レベルアップして昇進
- 実績バッジのコレクション
- 章ごとの正答率表示

## 🎓 学習システム

### レベルシステム
- レベル1: 新人（初期状態）
- レベル3+: 主任
- レベル6+: 課長
- レベル10+: 部長

### 経験値獲得
- シナリオクリア: スコア × 2
- 試験対策モード: 正解 × 10
- 最低保証: 20経験値

### 実績バッジ
- 🎯 初陣: 最初のシナリオをクリア
- ⭐ Lv5達成: レベル5に到達
- 🌟 Lv10達成: レベル10に到達
- 🏆 全制覇: 全シナリオをクリア
- 💯 満点: スコア100点以上を獲得
- 👑 マスター: 部長に昇進
