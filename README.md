# CPP Master - 調達プロフェッショナル養成アプリ

> 勉強が苦手でも楽しくCPPｂ級に合格できる革新的学習アプリ

## 📱 アプリ概要

CPP Masterは、調達プロフェッショナル認定資格（CPP）ｂ級の合格を目指す学習者のために開発された、ゲーミフィケーション要素を取り入れたモバイルファーストの学習アプリです。

### 🎯 特徴

- **📚 デジタル教科書**: スマートフォン最適化された読みやすい教材
- **🎮 ゲーミフィケーション**: RPG要素で学習継続をサポート
- **🧠 AI学習支援**: 個人の弱点を分析し、最適な学習プランを提案
- **⚡ マイクロラーニング**: 5-10分の短時間学習セッション
- **📊 詳細な進捗管理**: 視覚的な学習データ分析
- **🌐 PWA対応**: オフライン学習、アプリライクな体験

## 🚀 主要機能

### 学習機能
- **デジタル教科書**: 19章構成の完全網羅教材
- **インタラクティブ問題**: 選択式、記述式、シナリオ問題
- **音声読み上げ**: 通勤時間の音声学習対応
- **ブックマーク**: 重要箇所の保存・検索機能

### ゲーミフィケーション
- **レベルシステム**: 学習進捗に応じたレベルアップ
- **バッジ獲得**: 各種達成条件でのバッジコレクション
- **クイズバトル**: 他ユーザーとのリアルタイム対戦
- **デイリーチャレンジ**: 毎日の学習目標設定

### 学習分析
- **弱点分析**: AI による苦手分野の自動特定
- **進捗ダッシュボード**: 詳細な学習統計表示
- **学習計画**: 試験日から逆算した最適スケジュール
- **パフォーマンス追跡**: 正解率、学習時間の詳細分析

### ソーシャル機能
- **ランキング**: 週間・月間学習ランキング
- **学習グループ**: 同僚・友人との学習チーム
- **進捗シェア**: SNSへの学習成果共有

## 🛠️ 技術仕様

### フロントエンド
- **HTML5/CSS3/JavaScript (ES6+)**
- **PWA**: Progressive Web App対応
- **レスポンシブデザイン**: スマホ・タブレット・PC対応
- **Service Worker**: オフライン機能、バックグラウンド同期

### 主要ライブラリ・API
- **Web API**:
  - Service Worker API (オフライン機能)
  - Push API (プッシュ通知)
  - Web Audio API (効果音・音声)
  - Intersection Observer API (スクロール最適化)
  - IndexedDB API (オフラインデータ保存)

### PWA機能
- **オフライン対応**: 学習コンテンツのローカル保存
- **アプリインストール**: ホーム画面への追加
- **プッシュ通知**: 学習リマインダー
- **バックグラウンド同期**: 進捗データの自動同期

## 📱 使用方法

### 初回セットアップ
1. ブラウザでindex.htmlを開く
2. PWAインストールプロンプトに従ってアプリをインストール（オプション）
3. 初期設定（通知許可、学習目標設定）を完了

### 基本操作
- **ナビゲーション**: 画面下部の5つのタブで機能切り替え
- **学習開始**: ホーム画面から「今日の学習を続ける」をタップ
- **問題演習**: 問題タブから各種モードを選択
- **進捗確認**: 進捗タブで詳細な学習データを確認

### ショートカット
- **上スワイプ**: 学習画面で次のページ
- **下スワイプ**: 学習画面で前のページ
- **長押し**: 重要箇所のクイックブックマーク

## 🎯 学習効率化のコツ

### 効果的な学習方法
1. **デイリーチャレンジ**: 毎日の小目標をクリア
2. **弱点集中**: AI分析による苦手分野の重点学習
3. **スキマ時間活用**: 5-10分の短時間学習セッション
4. **継続学習**: 学習ストリーク（連続日数）の維持

- **章完了系**: 各章100%完了
- **特別バッジ**: 模擬試験満点、早朝学習習慣
