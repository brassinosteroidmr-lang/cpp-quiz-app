# 調達マスターへの道

CPP b級試験対策アプリ - 実務シミュレーション型＋試験対策型の学習Webアプリ

## 🌐 アプリケーションURL

**https://brassinosteroidmr-lang.github.io/cpp-quiz-app/**

## ✨ 主な機能

### 🎯 実務シミュレーションモード
- **リアルな業務シナリオ**: 緊急調達、品質問題対応、コスト削減プロジェクトなど、実際の業務場面を再現
- **意思決定トレーニング**: 複数の選択肢から最適な判断を選び、その結果を即座にフィードバック
- **ストーリー進行**: 選択によって展開が変わる、没入感のある学習体験
- **3つのシナリオ**を収録（すべて最初から利用可能）

### 📝 試験対策モード
- **145問の本格的な試験問題**を収録
- **全11章に対応**（第1章〜第11章）
- **四択形式**で実際の試験に準拠
- **各問題に詳しい解説付き**
- **わかりやすい正解・不正解表示**
- 章ごとの学習進捗管理
- 正答率の記録・分析

### 📊 学習管理機能
- **詳細な学習統計**: 解答済み問題数、全体正答率、学習進捗を可視化
- **章別進捗管理**: 各章の挑戦回数と正答率を記録
- **弱点分析**: 学習結果に基づいた的確なアドバイス
- **学習履歴の保存**: ブラウザに進捗を自動保存

### 📱 PWA対応
- **オフライン対応**: 一度読み込めば、インターネットなしでも学習可能
- **インストール可能**: スマホ・PCのホーム画面に追加して、アプリのように使用
- **レスポンシブデザイン**: スマホ・タブレット・PCあらゆるデバイスに対応

## 🚀 使い方

### Webで使う
1. https://brassinosteroidmr-lang.github.io/cpp-quiz-app/ にアクセス
2. 「新しく始める」をクリック
3. モードを選択：
   - 🎯 **実務シミュレーション**：ストーリーで学ぶ
   - 📝 **試験対策モード**：問題演習で学ぶ
4. 学習開始！

### ローカルで起動
```bash
# Pythonの場合
python -m http.server 8000

# Node.jsの場合
npx http-server
```

ブラウザで `http://localhost:8000` にアクセス

### スマホでアプリのように使う
ブラウザのメニューから「ホーム画面に追加」を選択

## 💡 特徴

### 詳しい解説
各問題に以下を含む解説を用意：
- ✅ 正解の理由
- 📚 関連知識の補足
- 💡 覚え方のコツ
- 🎯 試験のポイント

### わかりやすい判定表示
- **正解時**：大きな緑色の✓マークと「正解！」
- **不正解時**：大きな赤色の✗マークと「不正解」
- シンプルでわかりやすい演出

### 学習効果を高める工夫
- 問題ごとの詳細な解説で理解を深める
- 章ごとの正答率で弱点を把握
- 学習統計で進捗を可視化
- 所要時間の記録で学習ペースを管理

## 🎓 学習システム

### 学習進捗の管理
- **解答済み問題数**: 全体で何問解いたかを記録
- **全体正答率**: すべての問題の正答率を表示
- **完了シナリオ数**: クリアしたシナリオの数を管理
- **学習進捗**: 全問題のうち何％学習したかを可視化

### 章別の進捗管理
- **挑戦回数**: 各章で何問解いたかを記録
- **正答数**: 各章で何問正解したかを記録
- **正答率**: 各章の理解度を％で表示

### 学習結果の分析
- **スコア別コメント**: 得点に応じた励ましとアドバイス
- **弱点の把握**: 正答率の低い章を特定
- **学習アドバイス**: 次の学習方針を提案
