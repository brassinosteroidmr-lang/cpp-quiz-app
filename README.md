# CPP B級 学習アプリ

CPP（調達プロフェッショナル）B級試験対策Webアプリ。演習問題200問＋教科書モード22ユニットを収録。

## 🌐 アプリケーションURL

**https://brassinosteroidmr-lang.github.io/cpp-quiz-app/**

## ✨ 主な機能

### 📝 演習問題モード
- **200問の試験対策問題**を収録（4セクション×50問）
- **四択形式**で実際の試験に準拠
- **難易度表示**（★〜★★★）
- **各問題に詳しい解説付き**
- セクション・難易度・未解答絞り込みに対応
- 正答率・学習履歴をブラウザに自動保存

### 🃏 フラッシュカードモード
- 苦手問題を素早く復習
- 「わかった／わからない」で仕分けして効率学習

### 📚 教科書モード
- 4セクション・22ユニットのテキストを閲覧
- 表・メモボックス・語呂合わせ・公式ボックスなど視覚強化レイアウト
- ユニット末尾から関連問題演習に直接ジャンプ可能

### 📊 学習管理機能
- 全体正答率・学習日数・解答済み問題数を可視化
- セクション別進捗管理
- 学習履歴のLocalStorage保存

### 📱 PWA対応
- **オフライン対応**: 一度読み込めばインターネット不要
- **インストール可能**: ホーム画面に追加してアプリとして使用
- **レスポンシブデザイン**: スマホ・タブレット・PCすべてに対応

## 🚀 使い方

### Webで使う
1. https://brassinosteroidmr-lang.github.io/cpp-quiz-app/ にアクセス
2. 学習モードを選択：
   - 📝 **演習問題**：セクション・難易度を選んで問題演習
   - 🃏 **フラッシュカード**：苦手問題を集中復習
   - 📚 **テキストで学ぶ**：教科書モードで読み込み学習

### ローカルで起動
```bash
# Pythonの場合
python -m http.server 8000

# Node.jsの場合
npx http-server
```

ブラウザで `http://localhost:8000` にアクセス

### スマホでアプリのように使う
ブラウザのメニューから「ホーム画面に追加」を選択

## 📁 ファイル構成

```
cpp-quiz-app/
├── index.html              # メインHTML（SPA、全ロジック含む）
├── questions-data.js       # 演習問題データ（200問）
├── textbook-data.js        # 教科書データ（22ユニット）
├── manifest.json           # PWA設定
├── sw.js                   # Service Worker
├── icon.svg                # アプリアイコン
├── parse_questions.py      # 問題データ再生成スクリプト
├── parse_textbook.py       # 教科書データ再生成スクリプト
├── remake/                 # 素材フォルダ（変換元Markdownファイル等）
└── README.md               # このファイル
```

### データの再生成

問題・教科書データを `remake/` フォルダのMarkdownから再生成する場合：

```bash
python parse_questions.py   # → questions-data.js を上書き生成
python parse_textbook.py    # → textbook-data.js を上書き生成
```

## 🛠️ 技術スタック

- **HTML5 / CSS3 / JavaScript (Vanilla)**: フレームワーク不使用のピュアSPA
- **PWA**: Service Worker + manifest.json
- **LocalStorage**: 学習履歴の永続化
- **Python 3**: Markdown→JavaScript変換スクリプト（開発用）
